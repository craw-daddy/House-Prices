Welcome to my House Price Prediction website.
This website is designed to let a user input some features about a house and use this information to get a predicted house price.

NOTE: Data can be entered either through the form available (above) on this webpage, or via the query string to the URL "~/predict". The parameter names to be used in the query string are the values shown on the form below. Note that "mszoning" is a categorical variable.

The value for "mssubclass" is a numeric value (although this is a categorical value!). In the training data these were all positive integer values which represented a categorical value, but any numeric (floating point value) is accepted. Similiarly, the value for "lotarea" should also be numeric.

A slight bit of error checking is performed as values are submitted, namely the values for "mssubclass" and "lotarea" are validated to be numeric. If they aren't, an error message is displayed to the screen.
